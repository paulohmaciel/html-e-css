Dentro dessa tag p temos outras duas tags (tags inline), que são: a tag i para ITÁLICO, a tag b para o BOLD, a tag u para underline, tag s para riscado, tag marked para grifado.

Existem ainda outros elementos textuais inline, como sup, sub, small, big, abbr.

E não menos importante, temos a tag a, que serve para criarmos links:

Link
Veja como ela é codada:
<a href="https://google.com" target="_blank" >Link</a >
